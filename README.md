Set of configurable Gulp tasks for use in Bitrix CMS templates.

## Установка

### Установка с использованием пакетных менеджеров

**npm**: 
```
$ npm install --save-dev gulp-bitrix-templates-toolkit
```
**yarn**:
```
yarn add -D gulp-bitrix-templates-toolkit
```
